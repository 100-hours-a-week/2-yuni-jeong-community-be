## 🌎 실행 방법
**1. git clone**
```bash
git clone https://github.com/100-hours-a-week/2-yuni-jeong-community-be.git
```

**2. 루트 디렉토리에 .env 파일 만들고 아래와 같이 작성**
```env
PORT=8080 # 사용할 포트 번호
SESSION_SECRET='secret-key' # 세션 시크릿 키
```

**3. 의존성 설치**
```bash
npm install
```

**4. 실행**
```bash
npm run start
```
